# LUXTON Telegram Bot

Бот для магазина украшений для магнитотерапии LUXTONE с поддержкой через темы в группе.

## Установка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/dmxpqhq6pk-dot/luxton.git
cd luxton
```

2. Создайте виртуальное окружение и активируйте его:
```bash
python -m venv venv
source venv/bin/activate  # На Windows: venv\Scripts\activate
```

3. Установите зависимости:
```bash
pip install -r requirements.txt
```

4. Убедитесь, что файл `welcome_image.png` находится в корне проекта

## Запуск

```bash
python bot.py
```

## Функционал

- Приветственное сообщение с картинкой при `/start`
- Две инлайн-кнопки:
  - "Дополнительные звенья" - создает тему в группе для обсуждения доп. звеньев
  - "Задать вопрос" - создает тему в группе для вопросов
- Автоматическая пересылка сообщений между пользователем и темой в группе
- Ответы от поддержки автоматически доставляются пользователю в бота

## Настройка группы

1. Добавьте бота в группу с ID `-4631893826`
2. Включите темы (Topics) в настройках группы
3. Дайте боту права администратора с возможностью управления темами
4. Аккаунт с ID `8222462689` должен быть в группе для ответов на запросы

## Требования

- Python 3.8+
- Группа Telegram с включенными темами (Topics)
- Бот должен быть администратором группы
